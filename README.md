# dotfiles-linux

Config files for Linux

## Compatibilities

- ArchLinux
- ArchWSL

## Requirements

- git
- make

## Usage

```Bash
$ git clone https://github.com/SuperYuro/dotfiles-linux ~/dotfiles-linux
$ cd dotfiles-linux/
$ make archwsl # For ArchWSL
$ make laptop  # For laptop
$ make desktop # For desktop
$ make cliconfig # Config file only
```
